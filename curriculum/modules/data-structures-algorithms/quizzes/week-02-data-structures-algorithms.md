# Data Structures & Algorithms: Complexity and Linear Structures Quiz

1. What is the time complexity of performing a linear search on an unsorted array of size n?
   - O(1)
   - O(log n)
   - O(n)
   Answer: O(n)

2. Which linear data structure provides O(1) time complexity for both enqueue and dequeue operations in a queue implementation?
   - Stack
   - Linked List
   - Array
   - Binary Tree
   Answer: Linked List

3. If you need to insert an element into a sorted singly linked list but you do not have a pointer to the previous node, what is the worst‑case time complexity of that operation?
   - O(1)
   - O(log n)
   - O(n)
   Answer: O(n)

4. Which statement correctly describes the space complexity of a singly linked list?
   - It requires O(1) additional space per element
   - It requires O(n) additional space per element
   - It requires O(n^2) additional space per element
   - It requires O(log n) additional space per element
   Answer: It requires O(1) additional space per element
