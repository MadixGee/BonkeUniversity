# Caching and Data Access Patterns Quiz

1. What is the primary purpose of caching in software systems?
   - To encrypt sensitive data
   - To reduce latency and improve performance by storing frequently accessed data
   - To manage user authentication sessions
   - To compress data for faster transmission
   Answer: To reduce latency and improve performance by storing frequently accessed data

2. Which data access pattern is best suited for scenarios where data is rarely modified but frequently accessed?
   - Write-heavy pattern
   - Read-heavy pattern
   - CRUD pattern
   - Batch update pattern
   Answer: Read-heavy pattern

3. What is a common challenge when implementing a caching layer?
   - Ensuring cache consistency across distributed systems
   - Increasing server load
   - Reducing code complexity
   - Simplifying database queries
   Answer: Ensuring cache consistency across distributed systems

4. Which caching strategy is most effective for time-sensitive data that must be refreshed periodically?
   - LRU (Least Recently Used)
   - Time-based expiration
   - Size-based eviction
   - Random eviction
   Answer: Time-based expiration
