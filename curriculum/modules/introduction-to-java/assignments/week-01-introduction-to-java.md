# Build a Simple Java Command-Line Greeting Application

Students will create a Java program that prompts the user for their name and prints a personalized greeting. This exercise introduces basic Java syntax, compilation, execution, and simple I/O using the Scanner class. By completing this assignment, learners will gain hands‑on experience setting up a Java development environment and writing their first Java class.

## Deliverables
- A Java source file named GreetingApp.java containing the main class and method
- A compiled bytecode file GreetingApp.class (or instructions to compile with javac)
- A README.md file explaining how to compile and run the program
- Optional: A simple unit test using JUnit to verify the greeting output

## Acceptance Criteria
- The program compiles without errors using javac GreetingApp.java
- When executed, the program prompts the user to enter their name and then prints 'Hello, <name>!'
- The README.md includes clear steps to compile and run the program on a typical command line
- If a unit test is provided, it passes when run with JUnit
- Code follows basic Java naming conventions and includes appropriate comments
