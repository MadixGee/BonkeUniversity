# Introduction to Java

## Learning Objectives
- Write a simple Java program with proper syntax and structure
- Understand and use basic Java data types and operators
- Implement control flow structures (if-else, loops)
- Create and manipulate Java classes and objects
- Apply object-oriented programming (OOP) principles
- Perform basic input/output operations in Java

## Why This Matters
Java is one of the most widely used programming languages in enterprise systems, Android apps, and large-scale software projects. Mastering its fundamentals enables learners to build scalable, maintainable applications and understand core concepts applicable to other languages.
