# Introduction to Java Quiz

1. Which of the following is a primitive data type in Java?
   - int
   - String
   - boolean
   - ArrayList
   Answer: int

2. Which loop in Java executes at least once before checking the condition?
   - for
   - while
   - do-while
   - if
   Answer: do-while

3. Which keyword is used to create an instance of a class in Java?
   - create
   - new
   - instantiate
   - define
   Answer: new
