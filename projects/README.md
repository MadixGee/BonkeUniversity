# Projects — Your Builds

This is where **your** code lives. Every module requires a working build here (Charter #2, #5).
The Reviewer reads these directories to produce code + architecture reviews, and the
PASS verdict on your build is what unlocks the next module.

```
projects/
  module-01-project/     # your first build
  ...
```
Nothing you don't build yourself belongs here (Charter #6).
