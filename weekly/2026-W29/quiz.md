# Foundational Principles for AI/Software Engineering Projects

1. Which architectural pattern is most effective for enabling asynchronous communication between microservices in a distributed system?
   - Singleton Pattern
   - Observer Pattern
   - Event-Driven Architecture
   - Factory Pattern
   Answer: Event-Driven Architecture

2. In test-driven development (TDD), what is the primary purpose of writing unit tests before implementing code?
   - To reduce code complexity
   - To ensure code meets requirements before writing it
   - To speed up the development process
   - To minimize the need for debugging
   Answer: To ensure code meets requirements before writing it

3. When designing a caching strategy for a high-traffic web application, which factor is most critical to balance?
   - Cache size vs. latency
   - Cache consistency vs. scalability
   - Cache expiration vs. data privacy
   - Cache complexity vs. developer time
   Answer: Cache size vs. latency
