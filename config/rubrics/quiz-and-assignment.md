# Rubric — Quiz & Assignment (starter; used by Grader)

Score each criterion 0-4. **The trade-offs criterion is mandatory (Charter #4).**

| Criterion | 0 | 2 | 4 |
|---|---|---|---|
| Correctness | wrong | mostly right | fully correct |
| Understanding (the "why") | pattern-matched, no reasoning | partial reasoning | derives from first principles (Charter #1) |
| Application | not applied | applied with gaps | cleanly applied to a real problem (Charter #2) |
| Trade-offs articulated | none | names alternatives | names alternatives AND justifies the choice (Charter #4) |
| Communication | unclear | adequate | crisp, senior-level |

Feedback must be actionable and Socratic: point at the gap, ask the guiding question, do not hand over the answer (Charter #6).
